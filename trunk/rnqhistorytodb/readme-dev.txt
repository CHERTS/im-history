RnQHistoryToDB
--------------

Плагин для хранения истории RnQ в базе данных MySQL, PostgreSQL, Oracle, SQLite.

---------------------
ОС:		Win2000/XP/2003/Vista/7
IM-клиент:	RnQ версии 1106 и выше
БД:		MySQL 4.0, 4.1, 5.0, 5.1
		PostgreSQL 7.1 - 8.3
		Oracle 8i - 11i
		SQLite 3
		Firebird 2.x
